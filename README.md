# Cradle of Love

A community-driven project dedicated to nurturing meaningful connections and spreading positivity.

## Features

- **Community**: Connect with like-minded individuals in a supportive environment
- **Growth**: Personal and collective development through shared experiences
- **Support**: Always there when you need guidance and encouragement

## Website

The website is built with:
- HTML5
- CSS3
- Vanilla JavaScript

## Getting Started

1. Visit the live website at: [https://tim07vista.github.io/Cradle-of-Love/](https://tim07vista.github.io/Cradle-of-Love/)
2. Or clone this repository and open `index.html` in your browser

## File Structure

```
├── index.html      # Main HTML file
├── styles.css      # Styling
├── script.js       # JavaScript functionality
└── README.md       # Documentation
```
